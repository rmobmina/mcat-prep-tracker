# mcat-prep-tracker

## Features

- 16-week structured study plan covering all MCAT sections
- Clickable checkboxes for every daily task
- Per-week progress bars + global overall progress bar
- Collapsible weeks to keep things tidy
- Progress saves automatically in your browser, no account needed
- Single self-contained HTML file, no installs or dependencies

## Usage

1. Download `index.html`
2. Open it in any browser (Chrome, Firefox, Safari)
3. Start checking off tasks — your progress saves automatically

> Progress is saved to the browser you use. Don't clear your browser cache or you'll lose your checkboxes. To preserve progress across devices, keep the file on the same machine/browser.

## Study Plan Structure

| Phase | Weeks | Focus |
|---|---|---|
| Phase 1 | 1–4 | Content review (Bio, Biochem, Gen Chem, Orgo, Physics, Psych/Soc) + daily CARS |
| Phase 2 | 5–8 | UWorld question bank + mixed practice |
| Phase 3 | 9–12 | Blueprint/Kaplan practice + AAMC full-lengths |
| Phase 4 | 13–16 | Targeted weak areas + final AAMC exams |
